I. Question: 
  Почему 1-й и 9-й элементы отличаются по размеру?
Answear: 
  Блочные элементы 1-й и 9-й являются дочерними элементами, по отношению к родительскому элементу container. И для этих блочных элементов переопределен параметр size. В связи с чем происходит изменения по высоте (height), ширине (width), внутреннему отступу (padding), внешнему отступу (margin).

Разъяснения:
1. Используется относительная еденица измерения em - это относительная единица измерения, равная размеру текущего шрифта.

3. По умолчанию 1em - это размер шрифта, который установлен браузером. У меня 16 px для браузера "chrome".
Или например в нашем случае, для родительского элемента (container) задан параметр size, который равен 1em - 16 px.

.container {
    --size: 1em;
    --color: green;
}

4. Происходит переопределение параметра size

.container > div:first-child, 
.container > div:last-child {
  --size: 2em;
}

Соответственно переменная size: 2em в себе содержит размер шрифта в два раза больше. Т.е. 16 * 2 = 32.

5. Прямые наследники родительского элемента container, имеют следующие стили:

.container > div {
  width: var(--size);
  height: var(--size);
  padding: var(--size);
  margin: calc(var(--size) / 10);
  background-color: var(--color);
}

В случае для дочерних блочных элементов 1 и 9, параметр size = 32. Соотвестсвенно:

- width = 32 px
- height = 32 px
- padding = 32 px
- margin - (32) / 10 = 3.2 px

6. Для других  дочерних элементов параметр size = 16 px. Соответственно размер меньше, чем у 1-го и 9-го элемента.

II. Task and questions.
  Добавить код, который установит элементам 4, 5, 6 дугой цвет (применяя переменную --color), для этого вы свободны менять HTML / CSS, но необходимо мотивировать аргументами ваше решение! Приветствуется так же альтернативные решения!

1. Создала класс (bg) для дочерних блочных элементов 4, 5 ,6. Изменив html документ. Воспользовавшись возможностями классов, я создала одинаковый класс для всех 3-х элеметов, что позволяет обратиться ко всем 3-м элементам в стилях.

.container .bg {
    --color: red;
}

2. Аргументация: В связи с тем, что дочерние элементы находятся не в начале или в конце html дерева. Чтобы применить стили для элементов, необходимо создать классы. Так как стили будут накладываться на все 3 элемента одинаковые. Можно воспользоваться возможностями стилизации (один ко многим) и создать один класс для всех троих элементов. После чего в стилях указать, что для класса (bg) параметр цвета (color) другой по сравнению с другими элементами.
